# AI Devs 4th Edition – Notes

## Introduction

These notes summarise key concepts from the AI Devs 4th edition course.

## Module 1 – LLM Basics

- Large Language Models work by predicting the next token.
- Temperature controls randomness in generation.
- Context window limits how much text the model can process at once.

## Module 2 – Tool Use

- Function calling lets models invoke external tools.
- Tools can be filesystem operations, APIs, or computation.
- The agent loop: send query → get tool calls → execute → feed results back.

## Summary

Mastering prompt engineering and tool integration enables building powerful AI agents.
