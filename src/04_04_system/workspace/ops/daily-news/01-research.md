# Phase 1 — Research & Write

## Agent

Ellie (research specialist).

## Steps

1. Read `_info.md` in this folder to get the topic list and source mapping.
2. For each topic:
   a. Use web search to find today's top 3–5 news items from the mapped source.
   b. Write a markdown note to `ops/daily-news/{date}/{topic}.md`.
3. Each note must have:
   - Frontmatter: `title`, `date`, `tags`
   - A markdown list of items, each with a one-line summary and `[source](url)` link.

## Rules

- Do NOT scrape full articles — search snippets are sufficient.
- 3–5 items per topic. Quality over quantity.
- Write each topic file as soon as it's ready. Do not batch.
- Use the exact path pattern: `ops/daily-news/{date}/{topic}.md`.
