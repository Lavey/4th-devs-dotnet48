# Person Template

Target: `world/people/{slug}.md`

## Required frontmatter

```yaml
---
title: "{full name}"
status: seed
tags: [people]
---
```

## Sections

### Who

Role, company, what they do.

### Context

How we know each other, shared history.

### Communication

Preferred channels (email, Signal, Slack, etc.), timezone if known.

### Notes

Interests, expertise, things to remember.
