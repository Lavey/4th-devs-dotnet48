# Project Template

Target: `craft/projects/{slug}.md`

## Required frontmatter

```yaml
---
title: "{project name}"
description: "{one-line mission}"
status: growing
tags: [{project tags}]
---
```

## Sections

### Mission

What this project aims to achieve.

### People

Who's involved — link to [[world/people/]] notes.

### Stack

Technologies, frameworks, infrastructure.

### Key links

Repository, deployment, documentation URLs.

### Status

Current state, recent milestones, next steps.

### Related

Links to knowledge notes, other projects, tools.
