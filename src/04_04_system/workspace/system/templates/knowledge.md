# Knowledge Template

Target: `craft/knowledge/{category}/{slug}.md`

## Required frontmatter

```yaml
---
title: "{topic title}"
description: "{one-line summary}"
status: seed
tags: [{topic tags}]
---
```

## Sections

### Overview

What is this? One paragraph explanation.

### Core concepts

Key ideas, terms, or components. Use a list.

### Relevance

Why does this matter? How does it connect to current projects or interests?

### Sources

Required. At least one URL or reference.

### Related notes

Links to other knowledge notes, projects, or tools that connect to this topic.
