# Idea Template

Target: `craft/ideas/{slug}.md`

## Required frontmatter

```yaml
---
title: "{idea title}"
status: seed
tags: [{relevant tags}]
---
```

## Sections

### Context

What sparked this idea? Where did it come from?

### Related area

Which project, topic, or domain does this connect to?

### Next step

One concrete action to move this forward (or "park" if it's just a seed).
