# Templates

Use these templates when creating new notes. Each template defines the required frontmatter and sections for its note type.

| Template | Target folder | When to use |
|----------|---------------|-------------|
| [[system/templates/idea]] | `craft/ideas/` | Quick thought capture |
| [[system/templates/knowledge]] | `craft/knowledge/*` | Research or learning note |
| [[system/templates/project]] | `craft/projects/` | Long-living project |
| [[system/templates/experiment]] | `craft/lab/` | Prototype or POC |
| [[system/templates/person]] | `world/people/` | Relationship note |
| [[system/templates/source]] | `world/sources/` | Information channel |
| [[system/templates/event]] | `world/events/` | Time-bound occurrence |
| [[system/templates/service]] | `world/tools/` | Tool, app, platform |
| [[system/templates/shared]] | `craft/shared/` | Published content |
