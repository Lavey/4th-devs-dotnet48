# Shared Template

Target: `craft/shared/{slug}.md`

## Required frontmatter

```yaml
---
title: "{content title}"
description: "{one-line summary}"
status: seed
tags: [{relevant tags}]
---
```

## Sections

### Format

Article, workshop, newsletter, video, etc.

### Audience

Who is this for?

### Key points

Core message and main takeaways.

### Sources

References and research used.

### Distribution

Where and how this will be published.

### Status

Draft, review, published, updated.
