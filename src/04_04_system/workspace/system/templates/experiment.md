# Experiment Template

Target: `craft/lab/{slug}.md`

## Required frontmatter

```yaml
---
title: "{experiment name}"
description: "{what you're testing}"
status: seed
tags: [{relevant tags}]
---
```

## Sections

### Goal

What are you trying to learn or prove?

### Approach

How you'll build it — tools, libraries, rough plan.

### Code

Where the code lives (repo link or inline snippets).

### Outcome

What happened? Did it work? What did you learn?

### Promotion path

If successful, where does this go? (project, knowledge note, tool)
